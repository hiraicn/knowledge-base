# 隐私政策 — 知识库助手

最后更新: 2026-06-25

## 概述

「知识库助手」是一款浏览器扩展,帮助用户将网页内容保存为本地知识库。我们高度重视您的隐私。

**核心原则:您的数据完全属于您,存储在您自己的设备上。**

## 数据存储

所有文章数据(标题、正文、标签、分类)存储在浏览器的 `chrome.storage.local` 中,保存在您的本地设备上。我们没有任何服务器来接收或存储您的内容。

## 数据采集

我们**不采集**任何用户数据。具体来说:

- 不收集个人信息
- 不收集浏览历史
- 不使用分析工具(无 Google Analytics、无埋点)
- 不发送使用数据到任何服务器
- 不包含任何第三方追踪代码

## 权限说明

| 权限 | 用途 | 是否传输数据 |
|------|------|-------------|
| `activeTab` | 获取当前标签页内容(用户点击「加入知识库」时) | 否 |
| `scripting` | 注入提取脚本到当前页面,提取正文内容 | 否 |
| `storage` | 将文章保存在浏览器本地存储中 | 否 |
| `unlimitedStorage` | 允许存储大量文章数据 | 否 |
| `contextMenus` | 右键菜单「加入知识库」 | 否 |
| `notifications` | 添加成功/失败的通知 | 否 |
| `tabs` | 打开知识库浏览页面 | 否 |
| `host_permissions: api.github.com` | 仅在用户主动点击「云端同步」时,与 GitHub Gist API 通信 | 仅用户主动触发时 |

## GitHub Gist 同步(可选功能)

云端同步是**完全可选**的功能,需要用户手动配置:

1. 用户需自行创建 GitHub Personal Access Token(仅 `gist` 权限)
2. Token 存储在 `chrome.storage.local` 中,仅发送至 `api.github.com`
3. 同步数据以私有 Gist(private gist)形式存储,仅用户自己可见
4. 我们不接收、不中转、不存储您的 Token 或同步数据
5. 用户可随时在 GitHub 删除 Gist 或撤销 Token 来停止同步

## 数据安全

- 文章内容仅存储在本地浏览器中,不上传到任何服务器(GitHub Gist 同步除外,且需用户主动配置)
- GitHub Token 仅用于与 GitHub API 通信,不发送到其他任何地方
- 扩展不包含任何第三方分析或追踪代码

## 数据删除

- 在知识库页面可删除单篇文章
- 卸载扩展将清除 `chrome.storage.local` 中的所有数据
- 如使用了 Gist 同步,可在 GitHub 上删除对应的 Gist
