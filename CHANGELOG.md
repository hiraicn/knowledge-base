# 更新日志

## v3.0.0 (2026-06-26)

**第三阶段：智能化**

本版本引入五项智能化功能，将知识库从被动的收藏工具升级为主动的知识管理平台。

### 新功能

- **自定义分类体系**：不再硬编码分类和关键词，在设置页面（侧边栏 ⚙️ 按钮）中可自由增删改分类及关键词，支持一键恢复默认。自定义分类纳入 Gist 同步范围。
- **TF-IDF 智能评分**：autoTag 算法从纯词频计数升级为 TF-IDF（词频-逆文档频率），结合标题 3 倍权重提升，分类和标签更加精准。
- **知识图谱可视化**：独立页面，基于 D3.js v7 力导向图，展示文章之间的关联关系。连线类型包括共同标签（≥2 个）、相同域名、引用链接。支持拖拽节点、缩放平移、悬停高亮、分类过滤、标签搜索。
- **自动摘要**：收藏文章时自动生成提取式摘要（基于句子位置权重 + 关键词密度评分），在文章列表和搜索结果中显示摘要预览。
- **双向引用**：收藏时自动提取文章中的链接，如果链接指向的文章已在知识库中，自动建立双向引用关系（references / referencedBy），在文章详情页展示引用和被引用列表。

### 改进

- Gist 同步新增 `kb-settings.json` 文件，同步自定义分类设置。
- 导出 .md 文件的 frontmatter 包含 summary、references、referencedBy 字段。
- 删除文章时自动清理相关引用关系。
- 命令行版（grow.js）与浏览器插件版保持功能同步。

### 技术变更

- 新增 `extension/graph.html` + `extension/graph.js`：知识图谱独立页面。
- 新增 `extension/lib/d3.min.js`：D3.js v7.9.0 本地加载（Manifest V3 CSP 不允许外部 CDN）。
- `extension/lib/taxonomy.js`：导出 `DEFAULT_TAXONOMY`，保持向后兼容。
- `extension/lib/kb-engine.js`：新增 `getTaxonomy()`、`saveTaxonomy()`、`resetTaxonomy()`、`computeIDF()`、`tokenize()`、`generateSummary()`、`extractReferences()` 等函数。

## v2.0.0 (2026-06-24)

**初始发布：无服务架构 + Gist 同步**

- 零配置浏览器插件，数据存储在 `chrome.storage.local`。
- 一键收藏：任意网页点击即存，自动提取正文、转 Markdown、智能分类打标签。
- 批量导入：支持从文件夹导入 .md 文件、拖拽导入。
- Wiki 界面：Wikipedia 风格，左侧分类目录树 + 右侧正文渲染。
- 全文搜索：输入关键词即时搜索标题和正文。
- 暗黑模式：一键切换明暗主题。
- GitHub Gist 同步：跨浏览器/设备同步，私有 Gist 存储，按 URL 智能合并去重。
- 命令行抓取：`node grow.js <url>` 批量抓取网页或微信公众号文章。
- 导入/导出：支持 .md 文件导入导出，YAML frontmatter 格式。
- 键盘快捷键：`/` 搜索、`←→` 切换文章、`ESC` 退出、`Ctrl+Shift+K` 收藏。
