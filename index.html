核心功能

为知识管理而生

从收藏到整理，从浏览到同步，一切都在浏览器中完成

⚡

一键收藏

在任意网页按 Ctrl+Shift+K 或点击插件图标，自动提取正文内容并转换为 Markdown 格式，也支持右键菜单操作。

🏷️

智能分类

基于关键词频率分析，自动将文章归入 14 个分类领域，并生成内容标签，无需手动整理，收藏即分类。

📖

Wiki 浏览器

Wikipedia 风格的浏览界面，左侧分类目录树 + 右侧正文展示，支持全文搜索、暗黑模式、标签筛选、相关推荐。

☁️

云端同步

通过 GitHub Gist 实现跨浏览器/设备同步，数据以私有 Gist 存储，仅你自己可见，不经过任何第三方服务器。

📂

批量导入

支持从文件夹批量导入 Markdown 文件，重装插件后可快速恢复知识库，也支持单文件导入和全量导出。

🔒

隐私优先

所有数据存储在浏览器本地，不采集用户数据，不包含追踪代码。云端同步是可选功能，完全由你掌控。

使用方式

三步开始你的知识库

零配置，安装即用

1

安装插件

从 Chrome/Edge 商店安装，或下载 ZIP 包手动加载。无需注册，无需配置。

→
2

一键收藏

在任意网页按 Ctrl+Shift+K，或右键选择「加入知识库」，自动提取并保存。

→
3

浏览知识库

点击「查看知识库」打开 Wiki 界面，搜索、分类浏览、标签筛选，随时查阅。
